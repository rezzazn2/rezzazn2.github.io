Pengertian Fungsi Kuadarat

Fungsi kuadrat adalah fungsi yang disusun oleh persamaan kuadrat berbentuk umum f(x) = ax² + bx + c, dengan a ≠ 0. Grafik fungsi kuadrat berbentuk non-linear dalam koordinat kartesius yaitu berupa parabola. Garis non-linear adalah istilah untuk garis tidak lurus dalam ilmu matematika. Contoh :

  • f(x) = x²
  • y = -2x²
  • f(x) = 2x² + x
  • y = 7x² + 2x + 3
  • y = -3x² + 3x + 1
  • f(x) = 3x² + 1
  • 2y = x² + 2x + 1
Rumus-rumus dalam Fungsi Kuadrat
  • Bentuk umum :  f(x) = ax² + bx + c, dengan a ≠ 0.
  • Titik potong dengan sumbu x :  f(x) = y = 0
  • Titik potong dengan sumbu y :  f(0) = a0² + b0 + c, x = 0
  • Diskriminan / D :  D = b²– 4ac.
  • Titik puncak/Ekstrim/Balik :  (xp,yp) = (-b/2a, -D/4a)
